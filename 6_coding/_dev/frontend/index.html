Вам предстоит написать программу, которая должна пройти все тесты. Ее цель - определить, что изображено на картинке.
Все просто, входные данные - картинка, закодированная в base64. Она будет передаваться программе через стандартный поток входа (stdin). Ответ необходимо вывести буквами в stdout. (одно слово)

Правда, на написание журейки у меня было не так много времени...
А еще я очень люблю Яндекс

Техническая информация

Ограничение по времени на каждый тест: 1 секунда
Глобальное ограничение по времени: 100 секунд
Окружение запуска: docker.io/library/python:3.8
Рекомендуемый способ запуска: python скрипт/бинарь под Debian 10
Ваша программа запускается примерно так: /bin/<your file>. Поэтому не забывайте про шебанг (подробнее)

Пример программы, которая даже проходит 1-й тест!

#!/usr/bin/env python
import base64

data = input()
